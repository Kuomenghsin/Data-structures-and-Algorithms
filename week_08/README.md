# Binary Tree
 > 儲存資料的方法
 
 > 增加搜尋效率
 
- 處理資料：seach、sort
- 儲存資料、資料結構：linked list、stack、queue、heap

#### Binary Tree points
- root：樹的頂端，只會有一個
   - 底下可長出子節點（child）
      > 若child的parent不是root，則此 parent＆child 稱之為子樹（subtree）
- Edge：連結點
- node：每個節點      
- subtree：
    - Siblings（手足）：同階層的多個child
    - leaf node：自己一個，沒有child
