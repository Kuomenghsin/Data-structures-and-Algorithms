# Red Black Tree

BST有可能會弱化成linked list，一旦弱化成linked list，tree的好處（查詢可在O(log n)解決）就消失了

- tree結構：愈balance就可在O(log n)搞定

- BST + rotation = Red Black Tree
  > rotation：旋轉，會回到balance
  >> balance：上一層沒塞滿不能塞下一層

#### Red Black Tree小遊戲

  > non-balance：第n層還沒塞滿就要塞n+1層
